# Instruction Builder

## Live

```
https://instructionbuilder.netlify.app/
```

## Description

Users can create structured instructions for any hobby or project using a clean, neon-themed interface. The tool autosaves as users type, supports custom fields, and exports the full document to a styled PDF.

## Tech Stack

- Vite
- React
- JavaScript
- html2canvas + jsPDF for PDF rendering
